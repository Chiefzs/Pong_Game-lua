# Lua Pong 🏓

A classic Pong clone built with **Lua** and the **LÖVE (Love2D)** framework. This project was developed as a practice run to understand core game development concepts such as game loops, state machines, and physics mechanics.

## 🚀 Features
* **Classic 2-Player Gameplay:** Local multiplayer support.
* **State Machine:** Clean game states (`start`, `serve`, `play`, `done`).
* **AABB Collision Detection:** Accurate ball and paddle interactions.
* **Dynamic Difficulty:** The ball's speed increases by 5% with every paddle hit!
* **Score System:** First player to reach 10 points wins the match.

## 🎮 Controls

| Action | Player 1 (Left) | Player 2 (Right) |
| :--- | :---: | :---: |
| **Move Up** | `W` | `Up Arrow` |
| **Move Down** | `S` | `Down Arrow` |

* **Start / Serve / Restart:** `Enter` or `Space`
* **Quit Game:** `Escape`
